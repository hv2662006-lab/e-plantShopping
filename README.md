# e-plantShopping

e-plantShopping is a React application developed as part of the IBM Skills Network project.

## Project Overview
This application represents the Paradise Nursery plant shopping app. Users can navigate between the landing page, plants page, and cart page.

## Features
- Landing page with Get Started button
- Navigation bar with Paradise Nursery, Plants, and Cart options
- Cart page with total amount, continue shopping, and checkout buttons

## Technologies Used
- React
- Vite
- JavaScript
- CSS
